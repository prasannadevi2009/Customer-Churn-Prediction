# Customer Churn Prediction

## Project Overview
This project predicts customer churn using Machine Learning and visualizes business insights using Power BI.

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-Learn
- Power BI

## Dataset
Telecom Customer Churn Dataset
7043 Customers
33 Features

## Machine Learning Models
- Logistic Regression: 88.99%
- Random Forest: 91.34%

## Key Insights
- Month-to-month customers show higher churn.
- High monthly charges increase churn risk.
- Long-term customers are less likely to churn.
- Competitor offers are a major churn reason.

## Dashboard Features
- KPI Cards
- Churn by Contract
- Churn Reasons
- Payment Method Analysis
- Customer Retention Funnel
- Interactive Slicers
